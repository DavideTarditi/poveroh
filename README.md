<!-- Run Locally -->

## :running: Run Locally



Clone the project

```bash
  git clone https://github.com/DavideTarditi/poveroh.git
```

Go to the project directory

```bash
  cd poveroh
```

Install dependencies

```bash
  npm install
```

Install dependencies

```bash
  npm install
```
